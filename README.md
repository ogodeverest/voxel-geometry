## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/voxel-geometry.git
```

Go to the project directory

```bash
  cd voxel-geometry
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** Three.js
